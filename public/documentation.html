Movies API

An API for an application called “myFlix” that interacts with a database that stores data about different movies

Documentation of Endpoints

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Allow to create a new username, that is: allow users to register /users POST (Create) A JSON object holding data about the user, particularly the user's name that is being registered A JSON object holding data about the user, particularly the user's name that has been added. The uuid creates a new unique ID for the new user.
Allow to update their username /users/:id PUT (Update) A JSON object holding data about the user, containing user's new name that is being updated A JSON object holding data about the user, containing user's updated name
Allow the user to add a movie to their favorits with just an info that it has been added /users/:id/:movieTitle POST (Create) None A text message that a movie has been added.
allows the user to delete a movie from their favorits with just an info that it has been deleted /users/:id/:movieTitle DELETE None Just a text message that the movie has been deleted from the favorites
allows the user to delete their user name /users/:id DELETE None Just a text message that the username has been deleted
Gets the list of ALL movies and their Data in JSON /movies GET (Read) None A JSON object holding the data about the movies (description, genre, director, image URL, whether it’s featured or not)
Gets the data in JSON of a chosen movie by title. /movies/:title GET (Read) None A JSON object holding the data about a movie chosen by title (description, genre, director, image URL, whether it’s featured or not)
Gets the data about a genre (description) by name (e.g., “Thriller”) /movies/movie/genre/:genreName GET (Read) None A JSON object holding the data about a genre by name and its description.
Gets the data about a director (bio, birth year, death year) by name; /movies/movie/directors/:directorName GET (Read) None A JSON object holding the data about a director by name and its description.
